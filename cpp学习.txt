变量存在的意义——方便管理内存空间。给一段内存空间取名，方便管理这段空间
变量创建的语法：数据类型 变量名 = 变量初始值；
int a = 10;

常量
#define 宏常量：语法：#define 常量名 常量值
const修饰的变量：语法：const 数据类型 常变量名 = 常量值

关键字：C++中预先保留的单词
例如 int
不要用关键字给变量或者常量取名称

标识符命名规则
1、标识符不可以是关键字
2、只能包含字母数字下划线
3、第一个字符必须是字母或者下划线
4、标识符中的字母区分大小写
给变量起名的时候，最好能够做到见名知意

数据类型
C++ 规定在创建一个变量或者常量时。必须要指出相应的数据类型，否则无法给变量分配内存。
short 短整型 2字节 16位 (-2^15~2^15-1)
int 整型 4字节
long 长整型 windows 4字节 Linux 4字节或8字节
long long 长长整型 8字节
数据类型存在的意义，给变量分配合适的空间
整型表示整数

关键字 sizeof
利用sizeof可以统计数据类型所占的大小
语法 sizeof(数据类型/变量)		sizeof(int)

浮点型（实型）
单精度 float 4字节 7位有效数字
双精度 double 8字节 15~16位有效数字
cpp默认小数是双精度		3.14f这是单精度小数
控制台默认显示6位有效数字
科学计数法		float f2 = 3e2; //300		float f2 = 3e-2; //0.03

字符型
语法 char ch = 'a';
注意用单引号，不可以是双引号，单引号只能有一个字符，不可以是字符串
c和cpp中字符型只占1字节
字符型变量内部存储的对应的ASCII码

换行符\n

字符串型
c语言风格字符串：char 变量名[] = "字符串值";		[]的作用是数组化，最后一位系统自己加\0
c++风格字符串： string 变量名 = "字符串值";		注意包含一个头文件 #include <string>		字符串变量

布尔类型 bool 占用1字节
作用数据类型代表真 1 true 或者 假 0 false
bool类型  非零的值都代表真

运算符
1、算数运算符
除法：int n1 = 10; int n2 = 3;	则n1/n2 = 3	整数除法小数位会舍弃
取模（取余）：10%3 = 1;模3运算	只有整型可以取模运算，两个小数不能取模运算
前置递增运算符 ++a	//先+1，再表达式运算
后置递增运算符 a++	//先表达式运算，再让变量+1
int a = 10;
int b = a++ * 10;		答案 100

int a = 10;
int b = ++a * 10;		答案 110
